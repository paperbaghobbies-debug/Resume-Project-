🔒 How this sandbox actually works

This isn't a fake demo — it's a real, live payment flow running in Stripe's test mode. No real money is ever involved, no matter who clicks it.

What's actually happening

  1. "Test Stripe Webhook Payment" calls a serverless function that creates a genuine Stripe Checkout Session.
  2. You're redirected to Stripe's real hosted checkout page.
  3. After payment, Stripe independently calls a webhook — a server-to-server request your browser never sees — to confirm the payment happened.
  4. That webhook is cryptographically signature-verified, so it can't be spoofed.
  5. Only after that server-side confirmation does the "Premium Inspector" role get unlocked. Editing the URL or browser state can't fake this.

Try it yourself

Click "Test Stripe Webhook Payment" and use Stripe's official test card:

  • Card number: 4242 4242 4242 4242
  • Expiry: any future date (e.g. 12/30)
  • CVC: any 3 digits
  • ZIP: any 5 digits

After paying, you'll be redirected back here and should see "Authorized via Stripe" within a couple of seconds. Click "Verify Payment Status" afterward to re-check your access level anytime.
